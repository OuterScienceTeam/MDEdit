<h1>About MDEdit</h1>

<p><strong> Version 0.3-beta </strong></p>

<p>For current version and source code see &lt;<a href="https://github.com/outerscienceteam/mdedit">https://github.com/outerscienceteam/mdedit</a>&gt;</p>

<p>MDEdit is a Markdown editor written in C++ using Qt toolkit.</p>

<p>MDEdit was developed upon a friend&rsquo;s request with the goal<br/>
 of providing a user-friendly, cross-platform, light-weight Markdown editor.</p>

<p>MDEdit uses the Discount Markdown parser developed by David Loren Parsons<br/>
&lt;<a href="https://github.com/Orc/discount">https://github.com/Orc/discount</a>></p>
